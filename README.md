# Papatya Dental AI Proposal

Next.js App Router, Tailwind CSS ve Framer Motion ile hazırlanmış tek sayfalık proposal portfolyo.

## Geliştirme

```bash
npm install
npm run dev
```

## Kontroller

```bash
npm run lint
npm run build
```

## Deploy

Repo doğrudan Vercel'e bağlanabilir. Framework ayarı otomatik olarak `Next.js` olarak algılanır.
